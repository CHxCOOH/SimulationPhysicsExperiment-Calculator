分享几个实验的数据处理小程序。具体保留有效数字位数自行参照要求，一般为3~4位有效数字。
实验平台地址：http://118.184.217.73/

### 使用方法：

（C++）在可执行文件的目录下，将仿真实验测得数据存入in.txt文件中。各数据间以空格分隔。
输入数据格式可参考解压所得的in.txt数据，用实际测得数据覆盖即可。in.txt保存完毕后，运行exe文件，秒出结果。
